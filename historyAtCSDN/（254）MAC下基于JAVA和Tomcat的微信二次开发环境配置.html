自学时候，配置环境最麻烦。

第1步：

下载JDK1.7版本，完成后根据提示点击下一步进行安装。
JDK1.7下载地址：
链接: http://pan.baidu.com/s/1bRO8P4 密码: 9qif
默认安装地址：/Library/Java/JavaVirtualMachines/jdk1.7.0_79.jdk/Contents/Home/
安装完成后，任意开一个终端，输入java -version用于检测JDK是否安装成功，显示java version “1.7.0_79”即为成功。

第2步：

下载Eclipse Java EE for Mac，完成后解压缩就安装成功了。
Eclipse for Mac下载地址：
链接: http://pan.baidu.com/s/1gfHfpMb 密码: faij

第3步：

下载Tomcat7，完成后解压缩，我解压缩到了/usr/local路径下，这个可以自己掌握。
Tomcat7下载地址：
链接: http://pan.baidu.com/s/1geOqAx9 密码: 3fb1
打开终端，目录切换至/usr/local/apache-tomcat-7.0.70/bin，输入sudo sh startup.sh，启动Tomcat。可通过打开浏览器，地址栏中输入http://localhost:8080查看Tomcat是否已经启动。若要停止服务器，就在上述路径下输入sudo sh startup.sh
小提示：Mac下界面操作时，有可能找不到系统目录，只能找得到用户目录。这时候一直使用command+向上方向键一路往上切换路径，就可以找到了。实在是找不到的话，可以直接复制路径。

第4步：

下载Tomcat for Eclipse插件，并解压缩至Eclipse安装路径下的plugins目录下，即可实现在Eclipse的偏好设置中配置Tomcat服务器的功能。只用把解压缩后的文件夹放进去就好。
Tomcat for Eclipse插件下载地址：
链接: http://pan.baidu.com/s/1qYcRwMo 密码: naxy

第5步：

配置JAVA HOME环境变量
任意打开一终端，输入命令open .bash_profile,然后将

JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk1.7.0_79.jdk/Contents/Home
PATH=$JAVA_HOME/bin:$PATH

export JAVA_HOME
export PATH


代码插入到该文件中，保存，退出。

第6步：

至此，所需软件已经全部安装成功。此时，启动Eclipse之后，开始着手配置Tomcat。eclipse->preferences,找到Tomcat选项：选择Tomcat版本7.x，此时只是确定了服务器所采用的tomcat信息，仍没有创建出服务器。

第7步：

建立服务器。在与tomcat同级菜单中，找到Server->Server runtime environment，新建一个Server，最后OK，就完成了Tomcat和Eclipse的设置。正常的话，可以看到界面多了三个Tomcat按钮。没有看到的话，选择菜单Window ->Custmize Perspective，在Tomcat上打勾。
