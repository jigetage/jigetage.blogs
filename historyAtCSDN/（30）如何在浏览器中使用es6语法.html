1，npm install parcel-bundler -g 安装打包工具parcel-bundler

2，npm config set registry=https://registry.npm.taobao.org 切换npm镜像为淘宝，加快依赖包安装速度

3，npm config list 查看npm镜像是否切换成功

4，npm install parcel-bundler -g 切换镜像后，再次安装parcel-bundler

5，parcel 运行parcel获取当前路径下文件和文件夹

6，parcel index.html 开始打包，并在当前目录下开启服务，默认端口号是1234
