mysql5.7以后mysql太坑了，为了安全机制，会生成一个默认的root密码，在安装完毕后，运行mysql_secure_installation时，需要输入默认密码。但是，你找遍了整个mac，也找不到保存这个密码的文件放在了哪里。如果通过下载dmg安装包的形式，则不会碰到这个问题，因为安装包上会明确的写出密码。

那么，如果碰到了这个问题怎么办呢？

一般思路就是找密码，但是你会发现，找遍这个mac也找不到。

另一种思路是，在安全模式下运行mysql，重置初始密码，这种方法可以（网上有很多教程），但是太麻烦了。

偶然间发下了第三种解决办法。运行如下两行命令即可：

sudo chown -R _mysql:mysql /usr/local/var/mysql

sudo mysql.server start

运行完毕之后，你再运行mysql_secure_installation就不会提醒输入默认密码了，此时可以直接进行配置了。
