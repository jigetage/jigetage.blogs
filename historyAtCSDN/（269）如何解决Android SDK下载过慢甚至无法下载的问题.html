两种方案：

1，采用Android SDK Manager下载。

这是由于某种（墙的原因占绝大部分）原因，Android SDK Manager在国内无法下载，这里介绍一种不需要翻墙的解决办法。

第一步，修改C:\Windows\System32\drivers\etc下的hosts，以文本文档形式打开，在文档最后添加一句74.125.237.1 dl-ssl.google.com

第二步，然后重新启动Android SDK Manager即可。 

你会看到下载速度有质的提升。

2，Google网站下载含SDK版本的Android Studio，下面给出下载链接：

https://dl.google.com/dl/android/studio/install/1.0.1/android-studio-bundle-135.1641136.exe
