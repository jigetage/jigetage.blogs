docker也太强大了吧，竟然可以把本地的应用程序制作成docker镜像。。。

以nginx举例，方法如下：

比如，要从本地文件系统导入一个镜像，先下载了一个 ubuntu-14.04 的镜像，之后使用以下命令导入：

sudo cat ubuntu-14.04-x86_64.tar.gz | docker import - ubuntu:14.04
或者
sudo docker import ubuntu-14.04-x86_64.tar.gz ubuntu:14.04

然后查看新导入的镜像。

docker images
