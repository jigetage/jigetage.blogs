网上搜索了很多方法来解决安装go扩展失败的问题，很多回答都比较乱，自己按照下面的步骤解决了问题。

// 路径下创建此文件
mkdir -p $GOPATH/src/golang.org/x
// 切换到此目录
cd $GOPATH/src/golang.org/x      
// 通过git安装 tools
git clone https://github.com/golang/tools.git  
// 以golint举例：
安装 lint
git clone https://github.com/golang/lint.git    
// 然后执行
go get golang.org/x/lint/golint       

完成上述操作后，重启vscode即可，此时golint已经起作用了，会有诸如代码提示、补全等友好功能。

对于其余的比如gocode godef等的安装，可以采用重启vscode，然后根据提示install/install all的方式，或者通过command + shift + p打开palette，输入go install后，选择需要的go扩展，然后安装的方法。
